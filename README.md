# leaflet-challenge
Leaflet Map with Legend
========================

This script creates a Leaflet map with a legend that displays the depth of markers on the map.

Features
--------

* Displays a Leaflet map with a legend in the bottom right corner
* The legend shows the depth of markers on the map, with different colors indicating different depth ranges
* The legend has a white background with padding, rounded corners, and a slight shadow effect to make it stand out from the map

Usage
-----

1. Include the Leaflet library in your HTML file:
```html
<link rel="stylesheet" href="https://unpkg.com/leaflet@1.7.1/dist/leaflet.css" />
<script src="https://unpkg.com/leaflet@1.7.1/dist/leaflet.js"></script>
